2048 game

Files:
- index.html — main page (open in browser)
- styles.css — styling for the game
- game.js — game logic (Snake)

How to run:
1. Open `index.html` in any modern browser (Chrome, Edge, Firefox).
2. Use arrow keys (← ↑ → ↓) or the on-screen buttons to control the snake.
3. Click Start to begin, Restart to start over.
