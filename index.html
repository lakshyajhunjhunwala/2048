2-0-4-8 Game

Swipe or use arrow keys to slide tiles. Combine tiles with the same number to create larger ones!

Score: 0

How to Play

  • Use arrow keys or mobile buttons to slide tiles.
  • When two tiles with the same number touch, they merge into one.
  • Each merge adds the combined value to your score.
  • Game ends when there are no more possible moves.
